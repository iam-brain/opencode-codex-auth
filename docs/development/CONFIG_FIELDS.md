# Config fields

## Debug

Any of these enable debug logging:

- `CODEX_AUTH_DEBUG=1`
- `OPENCODE_OPENAI_AUTH_DEBUG=1`
- `DEBUG_CODEX_PLUGIN=1`

## Proactive refresh

- `OPENCODE_OPENAI_MULTI_PROACTIVE_REFRESH=true`
- `OPENCODE_OPENAI_MULTI_PROACTIVE_REFRESH_BUFFER_MS=<number>`

See `lib/config.ts`.
